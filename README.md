# 现代设计工具打造高效 SaaS 着陆页

## 项目描述
运用现代设计工具（如 Tailwind CSS、帧动画等技术）构建高效的 SaaS 着陆页。

- **框架**：Next.js、React
- **样式**：Tailwind CSS
- **动画**：Framer Motion（帧动画）
- **目标**：实现响应式、高性能的落地页设计
